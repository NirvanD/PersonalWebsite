Over the years, I have worked my many projects. Some, were for fun that I made by myself. Some, where a group effort. These projects range from a simple, yet complicated, fully responsive website to a python project that uses object detection.

Created a fully responsive website from scratch using HTML, CSS, and JavaScript for my robotics team in middle school.

Collaborated with a group to create a React Native app that can collect scouting data from competitions. The data is stored locally in an SQL database so that it can be anlyzed through analytical pages. Finally, data can be transfered wirelessly through bluetooth and exported in a CSV format.

Collaborated with a group to create a React Native app that can help inform users of upcoming and active natural disasters. This project uses MongoDB as a backend database and uses a whether API to gather information about natural disasters. Finally, this app has a chatbot feature to help comfort the user and answer any questions they may have.

This app was also built with React Native and was made for bedridden patients. This app was part of a larger project and includes a system where bedridden patients can easily recieve care in an outpatient clinic. The purpose of this app is to help patients schedule appointments with their physicians and provide an EMS vehicle tracker for the patient.

This project was built using HTML, CSS, JavaScript, Python, and a framework called Flask. XrayVision AI allows the user to input any symptoms they may be having. The code then compares these and displays the chance of having a certain disease. Then, the user can input an X-Ray scan. Finally, the code uses object dectection to detect if the patient has the disease. Currently, this project can detect up to 8 different diseases.
